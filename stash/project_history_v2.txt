PART 2: ARCHITECTURAL MILESTONES (COMPLETED)

[1. NATIVE TAURI DESKTOP CLIENT]
- Migrated terminal layouts to a glassmorphic React design system (matte black backdrops, 26px rounded corners, spacing boundaries, subtle ambient backlights).
- Implemented global hotkey intercepts: Ctrl+H toggles the HUD dashboard even when terminal text streams are active.

[2. PER-SESSION ISOLATED SANDBOXES]
- Created sequential, numbered container allocator naming ('academy-sandbox-1', 'academy-sandbox-2', etc.) inside the Tauri Rust library.
- When a user launches a desktop session, the backend checks active containers via SSH, allocates the next available index N, and creates/spawns the remote environment on Aeza VPS (89.22.239.107).
- Integrated Tauri window destruction handlers (tauri::WindowEvent::Destroyed) and docker '--rm' arguments. Closing the desktop app window automatically stops and prunes the remote container session.

[3. AI COMMAND OBSERVER LAYER]
- Before-Execution: intercepting XTerm keys on 'Enter' reads the typed command buffer, flags dangerous structures (e.g. 'rm -rf /'), and prints a red warning banner.
- After-Execution: buffers stdout streams, strips ANSI styling codes, detects prompt returns, and queries Gemini 2.5 Flash to generate real-time Socratic feedback based on execution outcomes.
- Sandbox Recycler: typing 'reset-sandbox' or 'reset' in XTerm intercepts the input, displays a status message, and runs a remote reboot sequence restoring the default container snapshot.

[4. GUEST ACCESS & API FALLBACKS]
- Saved default API key in public/config.json (safely ignored in gitignore). If a student has no personal key in local settings, the client falls back to this config, enabling unregistered sessions to start immediately.

[5. PREMIUM COSMETICS & ANIMATIONS]
- Animated Canvas: floating amber/orange particles drift smoothly in the background.
- Pulsating Input: Gemini-style neon glows continuously around the input field.
- Send Button: custom border, tap/hover Framer Motion spring micro-animations.
- Highlight Animation: animated glowing borders on active syllabus nodes.

PART 3: RUNTIME OPERATIONS & ONE-COMMAND LAUNCHER

We have created an automated launcher script 'start.sh' in the root folder. If you ever need to start the application when the assistant is offline:
1. Open terminal on your MacBook.
2. Navigate to root folder and run:
   ./start.sh
This automatically clears port 5173 to prevent HMR socket conflicts and launches the entire Tauri dev environment with a single command.
