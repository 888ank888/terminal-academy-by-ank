# Terminal Academy

An international, systems-first educational platform teaching real Linux administration, network architecture, and DevOps to adolescents through a live command-line interface.

## Repository Structure
- `/web-landing`: High-fidelity minimalistic landing page interface.
- `/cli-client`: Native Python terminal client built using `curses`.

## Technical Stack
- Frontend: Vanilla HTML5 / CSS3 (Linear-style retro-futuristic aesthetic)
- Core Client: Python 3.10 / Curses Library
- Sandbox: Lightweight Containerized Environments
